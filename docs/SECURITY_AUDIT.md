## 4. 安全建议

### 4.1 高优先级

| 建议 | 说明 |
|------|------|
| 生产环境启用 HTTPS | 使用 Nginx/Caddy 反向代理 |
| 限制 CORS 来源 | 生产环境限制为特定域名 |
| 定期更新依赖 | 每月运行 `pip-audit` |

### 4.2 中优先级

| 建议 | 说明 |
|------|------|
| 添加请求日志 | 记录所有 API 请求 |
| 实施 IP 限制 | 限制异常 IP 的访问 |
| 添加 API Key 过期 | 定期轮换 API Key |

### 4.3 低优先级

| 建议 | 说明 |
|------|------|
| 添加 WAF | Web 应用防火墙 |
| 实施 DDoS 防护 | 云服务 DDoS 防护 |
